# 版本记录

从 2026-04-26 起，项目版本重新编号。当前项目状态作为新的第一版基线，不再沿用之前的临时迭代编号。

## v1.1.0 - 成长任务中心与打卡体系增强版

- 时间：2026-04-26
- 定位：增强闽食成长岛、幼习宝班级成长任务、阅读打卡、老师活动卡、家长居家延续和奖章升级
- 状态：已通过 `npm run lint`、`npx tsc --noEmit`、`npm run build`
- 是否已同步服务器：未同步

### 本次主要改动

- 闽食成长岛新增“闽食小列车”，通过站点口令、短儿歌和美食站卡认识泉州美食。
- 闽食成长岛新增“美食猜猜乐”，通过宝箱线索猜食材，强化食材发现和小小播报。
- 幼习宝图卡升级为“幼习宝成长任务地图”，卡片可点击、可语音提示、可完成打卡。
- 幼习宝新增“阅读小书虫打卡”，支持听短绘本、说角色/画面/喜欢点和图书归位。
- 幼习宝新增“文明进餐操”，练习扶碗、坐稳、细嚼慢咽、按需取餐和餐后整理。
- 幼习宝新增“好习惯红绿牌”，用红绿牌判断进餐与生活行为是否合适。
- 儿童端移除旧“看图判断做法对不对”主入口，保留 `habitJudge` 历史记录兼容。
- 老师端新增 6 张“班级成长任务活动卡”，可带入活动课程方案草稿，老师确认后再生成。
- 家长端增强居家延续任务口径：阅读小书虫、家庭美食小管家、亲子尝新小挑战、睡前美食/故事小回顾。
- 成长记录增加奖章升级体系，按当前幼儿奖章或互动记录展示等级、下一等级差距和最近 3 枚奖章。
- AI prompt 增加闽食小列车、美食猜猜乐、阅读打卡、文明进餐操、红绿牌、儿歌口令、表扬语和家园同步约束。

## v1.0.0 - 第一版基线

- 时间：2026-04-26 17:55 +08:00
- 定位：童趣成长乐园第一版基线
- 状态：已通过 `npm run lint`、`npx tsc --noEmit`、`npm run build`
- 说明：此前零散改动归并为本基线版本，后续每次功能或文案调整都从小版本继续记录。

### 当前已纳入的主要能力

- 首页仅保留身份入口：儿童互动、教师辅助、家长端。
- 儿童端先选择幼儿小名牌，再进入幼习宝或闽食成长岛。
- 幼习宝包含习惯养成、安全判断、听绘本故事和语音播放能力。
- 闽食成长岛包含闽南食育互动、饮食偏好观察、餐盘营养提示和语音提示。
- 教师辅助台可管理花名册、查看游戏打卡汇总、调整小游戏文案、生成活动方案，并设置教师账号登录。
- 家长端通过幼儿身份查看老师同步内容、成长记录和反馈入口。

## 后续版本规则

- 小改动：递增补丁号，例如 `v1.0.1`、`v1.0.2`。
- 成组功能：递增小版本，例如 `v1.1.0`。
- 大幅重构：递增主版本，例如 `v2.0.0`。
- 每次改动后都在本文件追加：版本号、日期、改动点、验证结果、是否已同步服务器。

## 备份规则

- 当前基线源码压缩包放在本机 `version-backups/` 目录。
- 压缩包不包含 `node_modules`、`.next`、`.git`、本地密钥和运行日志。
- 后续需要回退时，优先按本文件找到版本号，再使用对应压缩包或 Git 提交记录恢复。
