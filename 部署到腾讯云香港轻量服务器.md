# 部署到腾讯云香港轻量服务器

这份文档适合当前项目的“国内访问更稳、先不备案”的部署路线。

## 为什么先选腾讯云香港轻量服务器

- 比 `vercel.app` 更适合中国大陆用户访问。
- 香港地域通常不要求 ICP 备案。
- 适合先给老师、家长、评审或少量用户稳定体验。

说明：
- 这里的“不要求 ICP 备案”是因为备案要求针对中国内地地域云资源。
- 如果以后要长期正式在中国内地开展网站服务，仍建议迁移到中国内地服务器并办理备案。

参考资料：
- 腾讯云轻量应用服务器产品页：<https://cloud.tencent.com/document/product/1207>
- 腾讯云轻量应用服务器 ICP 备案说明：<https://cloud.tencent.com/document/product/1207/45756>

## 推荐配置

建议先买：

- 地域：香港
- 系统：Ubuntu 22.04 LTS
- 配置：2 核 2G 或更高
- 带宽：4M 或更高

## 第一步：购买并登录服务器

购买后在腾讯云控制台拿到：

- 公网 IP
- root 密码

用 SSH 登录：

```bash
ssh root@你的服务器IP
```

## 第二步：开放服务器端口

在腾讯云轻量服务器防火墙里放行：

- `22`
- `80`
- `443`

## 第三步：安装 Node.js、Nginx、PM2

在服务器里执行：

```bash
apt update
apt install -y nginx curl git
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
npm install -g pm2
```

检查版本：

```bash
node -v
npm -v
pm2 -v
```

## 第四步：上传项目代码

如果服务器能访问 GitHub：

```bash
mkdir -p /var/www
cd /var/www
git clone https://github.com/HUAhua16888/-.git tongqu-growth-web
cd tongqu-growth-web
```

如果服务器也访问不了 GitHub：

- 可以把项目打包成 zip 手动上传到服务器。
- 解压到：

```bash
/var/www/tongqu-growth-web
```

## 第五步：配置环境变量

在服务器项目目录创建：

```bash
/var/www/tongqu-growth-web/.env.production
```

建议内容：

```bash
OPENAI_API_KEY=你的DeepSeek密钥
OPENAI_BASE_URL=https://api.deepseek.com
OPENAI_MODEL=deepseek-reasoner

NEXT_PUBLIC_SITE_URL=https://你的域名
NEXT_PUBLIC_SHOW_INTERNAL_NOTES=false
NEXT_PUBLIC_ENABLE_IMAGE_GENERATION=true
ENABLE_IMAGE_GENERATION=true
NEXT_PUBLIC_ENABLE_PREMIUM_TTS=true
NEXT_PUBLIC_TTS_VOICE_LABEL=小何 2.0
INTERNAL_HEALTH_TOKEN=一串只给自己用的随机口令

VOLCENGINE_ARK_API_KEY=你的火山密钥
VOLCENGINE_ARK_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
VOLCENGINE_ARK_IMAGE_MODEL=doubao-seedream-5-0-260128
VOLCENGINE_ARK_IMAGE_SIZE=2K
VOLCENGINE_ARK_VISION_MODEL=你的方舟视觉模型ID

VOLCENGINE_SPEECH_APP_ID=你的火山语音应用ID
VOLCENGINE_SPEECH_ACCESS_TOKEN=你的火山语音访问令牌
VOLCENGINE_SPEECH_SECRET_KEY=你的火山语音密钥
VOLCENGINE_TTS_ENDPOINT=https://openspeech.bytedance.com/api/v3/tts/unidirectional/sse
VOLCENGINE_TTS_RESOURCE_ID=seed-tts-2.0
VOLCENGINE_TTS_VOICE_TYPE=zh_female_xiaohe_uranus_bigtts
VOLCENGINE_TTS_AUDIO_FORMAT=mp3
VOLCENGINE_TTS_SAMPLE_RATE=24000
```

说明：
- 老师辅助页的内部备注默认隐藏，只有本地调试时才把 `NEXT_PUBLIC_SHOW_INTERNAL_NOTES` 改成 `true`。
- 如果视觉模型暂未开通，拍图分析会自动回退到基础观察卡。

## 第六步：安装依赖并构建

```bash
cd /var/www/tongqu-growth-web
npm install
cp .env.production .env.local
npm run build
```

## 第七步：用 PM2 启动

把项目里的 `ecosystem.config.cjs` 上传后执行：

```bash
cd /var/www/tongqu-growth-web
pm2 start ecosystem.config.cjs
pm2 save
pm2 startup
```

检查运行状态：

```bash
pm2 status
curl http://127.0.0.1:3000/api/health
curl -H "x-internal-health-token: 你的内部健康检查口令" http://127.0.0.1:3000/api/health
```

## 第八步：配置 Nginx 反向代理

把项目里的：

```bash
deploy/nginx.tongqu-growth-web.conf
```

复制到：

```bash
/etc/nginx/sites-available/tongqu-growth-web.conf
```

记得把：

```bash
your-domain.com
www.your-domain.com
```

替换成你自己的域名。

然后执行：

```bash
ln -s /etc/nginx/sites-available/tongqu-growth-web.conf /etc/nginx/sites-enabled/tongqu-growth-web.conf
nginx -t
systemctl restart nginx
```

## 第九步：绑定域名

在你的域名控制台添加解析：

- `A` 记录指向服务器公网 IP

如果有 `www`，再加：

- `www` -> 服务器公网 IP

## 第十步：配置 HTTPS

推荐用 Certbot：

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d 你的域名 -d www.你的域名
```

完成后你的网站就可以通过 HTTPS 访问。

## 稳定上线建议

当前项目建议先这样上线：

- 聊天功能开启
- 小游戏开启
- 老师辅助页开启
- 图片功能先关闭
- 语音先保留浏览器内置能力

这样对国内用户体验更稳。

## 后续升级顺序

建议后续按这个顺序升级：

1. 国内更稳的正式域名访问
2. PM2 + Nginx 稳定运行
3. 数据库存储聊天记录和勋章
4. 对象存储保存图片和音频
5. 后续再接更专业的语音识别与语音合成
